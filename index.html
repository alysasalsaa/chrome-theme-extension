How to Install
Chrome Web Store not required — install directly from source in 4 steps
1
Download the source
Click "Download from GitHub" → Code → Download ZIP → Extract the folder.
2
Open Chrome Extensions
Go to chrome://extensions in your browser address bar.
3
Enable Developer Mode
Toggle the Developer mode switch in the top right corner.
4
Load the extension
Click Load unpacked → select the extracted chrome-theme-extension-main folder. Done!
